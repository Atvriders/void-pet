# VOID

A browser-based Tamagotchi about a sentient data entity you found in a corrupted server.

Not a cute animal. An abstract geometric lifeform made of light and code that evolves through 5 stages, decays in real time, and can corrupt and die if neglected.

![Stage: Ascendant](https://img.shields.io/badge/stage-ascendant-blueviolet) ![Docker](https://img.shields.io/badge/docker-ghcr.io-blue)

## Gameplay

Care for your entity by managing four interconnected stats:

| Stat | Description | Danger |
|---|---|---|
| **Signal** | Hunger equivalent — feed it data | Drops to 0 in ~3.5h |
| **Coherence** | Mood — keep it engaged | Accelerated by low Signal |
| **Heat** | Bad when high — generated by actions | Accelerates all decay above 80 |
| **Power** | Energy — depleted by actions | Restores during hibernation |

Stats decay in real time based on elapsed time since last visit, so your entity needs genuine care across sessions.

## Actions

| Action | Effect | Cooldown |
|---|---|---|
| **Inject** | +Signal, +Heat | None |
| **Simulate** | +Coherence, +Heat, −Power | 30s |
| **Defrag** | −Heat, +Coherence | 60s |
| **Hibernate** | Toggle sleep — restores Power, cools Heat | None |
| **Overclock** | Big +Coherence, big +Heat, 20% chance of thermal surge | 120s |

## Evolution

Your entity evolves through 5 stages based on age and average stat quality:

```
Seed → Sprite → Entity → Apex → Ascendant
```

Each stage has a distinct canvas animation — more rings, particles, and complexity as it grows. Reaching Ascendant resets the entity but preserves your ascension count.

## Corruption

If Signal and Coherence both drop below 5 while Heat exceeds 85, the entity begins fragmenting. Run **Defrag** 3 times to recover. Neglect it long enough and it corrupts permanently — you can abandon it and start over.

## Run

```bash
docker compose up -d
```

Runs on **port 3003**. Requires the GHCR image to be available (built by CI on push to master).

To update:

```bash
docker compose pull && docker compose up -d
```

## Build locally

```bash
npm install
npm run dev     # dev server on :3003
npm run build   # production build
```

## CI

Pushes to `master` build and publish the Docker image to:

```
ghcr.io/atvriders/void-pet:latest
```
