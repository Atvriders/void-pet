# VOID

**You found a sentient data entity in a corrupted server. Keep it alive.**

VOID is a browser-based virtual pet for people who find Tamagotchis too cheerful. Your entity is an abstract geometric lifeform made of light and code — not a cute animal. It evolves through 5 stages, decays in real time, and can corrupt and die if neglected. Care is tracked on a device-local leaderboard. Your progress persists across sessions in your browser's localStorage.

---

## Features

- **Username system** — Enter a name at startup. Your entity's state is saved in your browser under that name. Returning with the same name on the same browser restores your entity. No accounts, no passwords.
- **Leaderboard** — A device-local leaderboard tracks top care scores across all usernames on this browser.
- **Pet face** — The entity has eyes and a mouth that reflect its current mood in real time. Watch the face — it often signals a problem before the stats do.
- **Mobile-friendly** — All controls are touch-accessible and the layout adapts to small screens.

---

## Gameplay

Your entity has four stats, each ranging from 0 to 100. They decay in real time while the tab is open. Stats do not decay while the tab is closed (active-play design).

| Stat | Description | Danger zone |
|---|---|---|
| **Signal** | Hunger equivalent — feed it data packets to keep it high | Below 20 accelerates Coherence decay |
| **Coherence** | Mood and stability — keep it engaged | Below 30 triggers `sad` mood; falls faster when Signal is low or Heat is high |
| **Heat** | Thermal load — generated by actions, bad when high | Above 80 accelerates Coherence and Power decay; triggers `overheating` mood |
| **Power** | Energy — consumed by actions, restored during hibernation | Below 15 triggers `tired` mood; minimum thresholds gate SIMULATE and OVERCLOCK |

Care score accumulates over time based on stat health: +1 point per minute for each stat in a healthy range (Signal > 60, Coherence > 60, Heat < 40, Power > 40). Maximum rate is 4 points per minute.

---

## Actions

| Action | Sub-label | Effect | Cooldown |
|---|---|---|---|
| **INJECT** | feed | +22 Signal, +4 Heat | None |
| **SIMULATE** | play | +25 Coherence, +18 Heat, −15 Power. Requires ≥15 Power. | 30 seconds |
| **DEFRAG** | water | −32 Heat, +8 Coherence. Required to recover from corruption. | 60 seconds |
| **HIBERNATE** | sleep / wake | Toggles rest mode — slower stat decay, faster Heat cooling, Power regeneration. Disables other actions. | None |
| **OVERCLOCK** | pet | +40 Coherence, +35 Heat, −25 Power. Requires ≥25 Power. 20% chance of thermal surge (+60 Heat instead). | 120 seconds |

All actions except HIBERNATE are blocked during hibernation. Actions except DEFRAG are blocked while the entity is corrupted.

## Evolution

The entity evolves through 5 stages based on age, average stat quality, and accumulated care score. Evolution is blocked during hibernation.

Stat average used for evolution thresholds: `(Signal + Coherence + (100 − Heat) + Power) / 4`

| Stage | Evolves From | Min Age | Min Avg Stat | Min Care Score |
|---|---|---|---|---|
| **Seed** | *(starting stage)* | — | — | — |
| **Sprite** | Seed | 15 min | 50 | 0 |
| **Entity** | Sprite | 60 min | 55 | 50 |
| **Apex** | Entity | 180 min | 60 | 200 |
| **Ascendant** | Apex | 480 min | 65 | 500 |

Each stage has a distinct canvas animation — more rings, particles, and visual complexity as the entity grows. Reaching **Ascendant** increments your ascension count. All stats, age, and care score carry forward — there is no reset on ascension.

## Moods

The entity's face and status label reflect its current condition. Evaluated top-down — first match wins.

| Mood | Label | Trigger |
|---|---|---|
| **corrupted** | ⚠ CORRUPTION DETECTED | Stage is `corrupted` |
| **sleeping** | HIBERNATING | Hibernation is active |
| **overheating** | THERMAL OVERLOAD | Heat > 80 |
| **hungry** | SIGNAL LOST | Signal < 20 |
| **tired** | LOW POWER | Power < 15 |
| **happy** | STABLE ◈ HARMONISED | Coherence > 70, Signal > 60, Heat < 40 |
| **sad** | COHERENCE DEGRADING | Coherence < 30 |
| **neutral** | NOMINAL | None of the above |

---

## Corruption and Recovery

If Signal < 5 **and** Coherence < 5 **and** Heat > 85 simultaneously for more than 20 consecutive seconds, the entity becomes **corrupted**.

To recover: run **DEFRAG 5 times**. Each DEFRAG reduces the remaining recovery count by one. When complete, the entity is restored to Seed stage with partial stats (Signal 40, Coherence 40, Heat 50, Power 40).

If you choose not to recover, you can abandon the entity and start fresh — but the leaderboard retains its record.

## Run

```bash
docker compose up -d
```

Runs on **port 3004**. Requires the GHCR image (built by CI on push to master).

To update:

```bash
docker compose pull && docker compose up -d
```

---

## Build Locally

```bash
npm install
npm run dev     # dev server on :3004
npm run build   # production build
```

---

## Stack

- **Frontend:** React 18 + TypeScript + Vite
- **Rendering:** HTML5 Canvas (HiDPI-aware, 480×480px, real-time RAF loop)
- **State:** `useReducer` + `localStorage` (no backend, no accounts)
- **Container:** Docker via GHCR (Nginx static serve)
